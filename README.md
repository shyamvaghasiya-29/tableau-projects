# Blinkit Store Analysis Dashboard

## Live Dashboard
[View Dashboard on Tableau Public](https://public.tableau.com/views/BlinkitStoreAnalysis/BlinkitDashboard)

## Project Overview
This Tableau dashboard provides a comprehensive analysis of Blinkit's sales performance, customer purchasing behavior, and product trends. The dashboard helps identify key factors influencing sales across different outlet types, locations, and product categories. Through interactive visualizations, users can explore revenue patterns, product performance, outlet efficiency, and customer preferences to support data-driven business decisions.

## Key Objectives
- Analyze total sales and revenue performance.
- Evaluate sales by product category and item type.
- Compare performance across different outlet sizes and locations.
- Identify top-performing products and outlets.
- Understand customer purchasing patterns and preferences.
- Generate actionable insights for improving sales and operational efficiency.

## Tools Used
- Tableau Public
- Excel/CSV Dataset
- Data Visualization
- Business Analytics

## Key Insights
- Total sales generated across all outlets.
- Best-performing product categories.
- Sales distribution by outlet type and location.
- Impact of outlet size on revenue generation.
- Customer demand trends across different products.
- Opportunities for inventory and sales optimization.
